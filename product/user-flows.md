# User Flows

Step-by-step journeys through the l00p app.

---

## üöÄ First-Time User Flow

**1. Download & Open**
- User downloads from App Store
- Opens app for first time

**2. Sign In**
- Choose: Sign in with Apple OR Google
- Tap to authenticate
- Wait for verification

**3. Profile Setup**
- Add photos (at least 1 required)
- Enter name and birthday
- Select gender
- Set home location (map picker)
- Write bio (optional but recommended)
- Choose themes/interests

**4. Preferences Setup**
- Select "Interested in" (gender preference)
- Set age range
- Set distance preference

**5. Start Discovering**
- Feed appears with first profiles
- Tutorial hints shown on first swipe

## üíï Matching Flow

**Finding a Match:**
1. Browse profiles in Feed
2. Swipe right on someone you like
3. If they also swiped right ‚Üí **Match!**
4. Chat unlocks automatically

**From Likes You:**
1. Tap "Likes" tab (heart icon)
2. See people who liked you (blurred for free users)
3. Swipe right ‚Üí Instant match
4. Chat opens

---

## üí¨ Conversation Flow

**Starting a Chat:**
1. Match appears
2. Go to Messages tab
3. Tap conversation
4. Type message or record voice note
5. Send

**Sending Voice Notes:**
1. Tap microphone icon
2. Hold to record (max 30s)
3. Release to preview
4. Tap send

---

## ‚òï Meeting Invite Flow

**Sending an Invite:**
1. Open chat with match
2. Tap coffee/venue icon
3. Choose venue type or search location
4. Select date/time (optional)
5. Add personal message (optional)
6. Send invite

**Receiving an Invite:**
1. Push notification arrives
2. Open chat
3. See invite card
4. Tap "Accept" or "Decline"

**After Accepting:**
1. Invite card updates to "Accepted"
2. Navigate button appears
3. Tap "Navigate" ‚Üí Opens Apple Maps
4. Get directions to venue

## ‚öôÔ∏è Settings Flow

**Editing Profile:**
1. Tap Profile tab
2. Tap any section to edit
3. Make changes
4. Changes auto-save

**Blocking a User:**
1. Open chat with user
2. Tap menu (three dots)
3. Select "Block User"
4. Confirm action
5. User removed from feed and chat

**Deleting Account:**
1. Go to Profile tab
2. Tap Settings gear icon
3. Scroll to "Delete Account"
4. Confirm deletion (requires tap)
5. Account permanently removed
