# l00p Documentation

Welcome to the l00p product documentation. This guide covers all app features and testing procedures.

## What is l00p?

**l00p** is an AI-powered dating app that connects people through meaningful conversations rather than endless swiping. Our AI companion, **Ghost**, learns your personality and helps you find genuine connections.

## Key Features

| Feature | Description |
|---------|-------------|
| **Smart Feed** | Discover matches with full-profile cards |
| **Ghost AI** | AI companion that learns your preferences |
| **Map Discovery** | Find people nearby in real-time |
| **Voice Messages** | Send voice notes up to 30 seconds |
| **Meeting Invites** | Coffee date scheduling built-in |
| **Story Loop** | Mini-games to break the ice |
